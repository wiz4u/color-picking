# Color Picking
*Let's look anew at the world around you*


**[Color Picking](http://wiz4u.github.io/color-picking/)** is a game
to look for the intended color using the camera.


## Supported Browser
- Chrome on Android (best :smile:)
- Chrome on Mac
- Firefox on Mac
